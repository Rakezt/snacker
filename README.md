## Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/snacker.git
   cd snacker
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm start
   ```

4. Open your browser at:

   ```
   http://localhost:3000
   ```
